True story

There are few ways to define (Є кілька способів визначити ) font-size:

  • px are good if you need strict definition for a size of text. In other cases it is better to use relative units listed below. px підходять, якщо вам потрібно чітке визначення розміру тексту.( В інших випадках краще використовувати відносні одиниці, наведені нижче.)
  • em based on current font size(em на основі поточного розміру шрифту)
  • rem based on(на основі) <html> font-size(розмір шрифту)
  • % based on parent font size
  • vh (viewport height) based on screen height ((висота вікна перегляду) на основі екрана висота)
  • vw (viewport width) based on screen width ((ширина вікна перегляду) на основі екрана ширина)
